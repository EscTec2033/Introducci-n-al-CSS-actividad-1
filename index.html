ACTIVIDAD 1

  1. ¿Qué es CSS?

    2. CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento HTML o XML. Permite separar el contenido (HTML) de la presentación (estilos).

  2. ¿Cuál es la función principal de CSS?

    3. La función principal de CSS es controlar el diseño y la apariencia visual de los elementos en una página web, como colores, fuentes, márgenes, espaciados y disposición de los elementos.

  3. ¿Cómo se puede incluir CSS en un documento HTML?
    • Inline: Usando el atributo style en las etiquetas HTML.
    • Interno: Dentro de una etiqueta style en la sección head del documento HTML.
    • Externo: En un archivo separado con extensión .css, que se vincula en la sección head usando la etiqueta link.
  4. ¿Qué son los selectores en CSS?

    5. Los selectores son patrones que se utilizan para seleccionar los elementos HTML que se desea estilizar. Ejemplos de selectores incluyen selectores de tipo, de clase, de ID y selectores de atributos.

  5. ¿Qué son las propiedades y valores en CSS?

    6. Las propiedades son características de los elementos HTML que se pueden modificar, como color, font-size o margin. Los valores son las configuraciones específicas que se asignan a esas propiedades, como red, 16px o 10px.

  6. ¿Cómo se aplica el modelo de caja en CSS?

    7. El modelo de caja describe cómo se representan los elementos en CSS. Cada elemento se considera una caja que tiene márgenes, bordes, rellenos (padding) y contenido. Se puede ajustar el tamaño y la posición de estas cajas usando propiedades como width, height, margin, padding, y border.

  7. ¿Qué es el "cascading" en CSS?

    8. El "cascading" se refiere al orden de prioridad que tienen los estilos cuando se aplican a un elemento. Los estilos pueden provenir de diferentes fuentes (estilos en línea, internos, externos) y se aplican según su especificidad y el orden en el que aparecen.

  8. ¿Qué son las media queries?

    9. Las media queries son una característica de CSS que permite aplicar estilos diferentes según las características del dispositivo que está visualizando la página, como el tamaño de la pantalla. Se utilizan comúnmente para crear diseños responsivos.

  9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?
    • Colores de fondo y texto.
    • Tipografía (fuentes, tamaños, estilos).
    • Espaciado (márgenes y rellenos).
    • Bordes y sombras.
    • Alineación (centrado, justificado).
  10. ¿Cómo se puede usar CSS para crear un diseño responsivo?

    11. Para crear un diseño responsivo, se pueden usar:

    • Media queries para aplicar diferentes estilos según el tamaño de la pantalla.
    • Unidades relativas (como porcentajes o em) para tamaños que se adaptan al tamaño del contenedor.
    • Flexbox y Grid para crear diseños fluidos que se ajusten a diferentes tamaños de pantalla.

Leon

El león (Panthera leo) es un mamífero carnívoro de la realeza familia de los félidos y una de las cinco especies del género Panthera.

Elefante

Los elefantes o elefántidos (Elephantidae) son una familia de mamíferos placentarios del orden proboscideos.

Tigre

El tigre (Panthera tigris) es una de las especies de la subfamilia de los panterinos pertenecientes al género Panthera.
